# Discord GuruGuruBot

## Setup settings

Create a settings file at the `src` folder contaning the following lines:

```json
{
    "discord": {
        "token": "discord-api-token"
    },
    "adapters": {
        "loop": {
            "channels": [
                "channel-id-1",
                "channel-id-2"
            ]
        }
    }
}
```
